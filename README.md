JACKHQ.com Website

## Development Setup

```
npm install
bower install
npm install w3 -g
```

## To run in development

```
npm run dev
```

## Notes

* The team members for the site are in the `app/team/team.js` file
* The team pics are in the `public/images` folder
* This website uses the VUE.js framework: http://vuejs.org/
