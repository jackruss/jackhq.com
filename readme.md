## Testing

Requires NodeJs > 0.3.7
Requires Coffee-Script 1.0.1
Requires Vows Latest
Requires Zombiejs

On a Mac

Install node and npm via
with npm install:

    npm install coffee-script
    npm install vows
    npm install zombie

To run the tests

    time vows test/*  --spec
